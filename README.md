# Aigei Audio DownloaderA Chrome extension that automatically detects and downloads audio files from aigei.com with robust IDM (Internet Download Manager) integration.## Features- **Automatic Detection**: Automatically detects audio files on aigei.com pages- **Badge Counter**: Shows the number of detected audio files on the extension icon- **Instant Download Prompt**: Displays a prompt when new audio files are detected with "Download Now" and "Later" options- **IDM Integration**: Seamlessly integrates with Internet Download Manager for optimal download performance- **Silent Downloads**: Downloads files without interrupting your browsing experience- **Simple UI**: Clean and minimalist popup interface## Installation### From Release (Recommended)1. Download the latest `aigei-audio-downloader.zip` from the [Releases](../../releases) page2. Extract the zip file to a folder on your computer3. Open Chrome and go to `chrome://extensions/`4. Enable "Developer mode" in the top right corner5. Click "Load unpacked" and select the extracted folder6. The extension is now installed and ready to use### From Source1. Clone or download this repository2. Open Chrome and go to `chrome://extensions/`3. Enable "Developer mode" in the top right corner4. Click "Load unpacked" and select the project folder5. The extension is now installed and ready to use## How to Use1. **Navigate to aigei.com**: Visit any page on aigei.com that contains audio files2. **Automatic Detection**: The extension will automatically detect audio files and show a badge counter on the extension icon3. **Download Options**:   - **Instant Download**: When a new audio file is detected, a prompt will appear in the top-right corner with "Download Now" and "Later" options   - **Manual Download**: Click the extension icon to open the popup and manually download detected files4. **Clear List**: Use the "Clear" button in the popup to remove all detected files from the list## Requirements- **Chrome**: Version 114 or higher- **IDM (Recommended)**: Internet Download Manager for optimal download performance- **Permissions**: The extension requires permissions to access aigei.com and manage downloads## Permissions Explained- `activeTab`: To detect audio files on the current tab- `downloads`: To initiate file downloads- `webRequest`: To intercept and analyze network requests for audio files- `storage`: To store detected audio file URLs- `scripting`: To inject content scripts for audio detection- `tabs`: To manage tab information and badge counters- `contextMenus`: For right-click context menu options- `declarativeNetRequest`: To filter and process network requests efficiently## Technical Details- **Manifest Version**: 3 (latest Chrome extension standard)- **Service Worker**: Uses modern service worker architecture for background processing- **Declarative Net Request**: Utilizes Chrome's latest networking API for efficient request filtering- **Host Permissions**: Limited to aigei.com for security and privacy## DevelopmentThis extension is built using:- Chrome Extension Manifest V3- JavaScript ES6+- Chrome Extension APIs- Declarative Net Request API## PrivacyThis extension:- Only works on aigei.com domains- Does not collect or transmit any personal data- Stores detected audio URLs locally in your browser- Does not track your browsing activity outside of aigei.com## SupportIf you encounter any issues or have suggestions:1.
